# 学生管理系统 接口文档

## 登录接口

> 根据关键字搜索结果，通过调整类型，搜索不同的值

**必选参数 :** 

1. `username` : 用户名
2. `password` : 密码

**接口地址：** /user/login

**请求方式：** post
